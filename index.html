☁ Google Sheets — Fonte de dados

Estrutura da planilha (linha 1 = cabeçalho):
nro | frase | categorias | tipo | imagem | notas

• Categorias separadas por ; → ex: Filosofia;Motivação
• Tipo: frase ou imagem
• O nro é o identificador único — nunca mude ele
• Sincronizar atualiza itens existentes pelo nro e adiciona os novos
No Sheets: Arquivo → Compartilhar → Publicar na web → Planilha1 → CSV → Publicar → cole o link completo aqui (o app extrai o ID automaticamente)
Última sync: nunca
